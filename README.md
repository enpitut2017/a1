## A1 「就活 from A1」
就活体験談を共有するサービス
## エレベーターピッチ
「自分が就活にあたって何をしたら良いかわからない問題」を解決したい「就活のビジョンが無い筑波大生」向けの「就活への臨み方」がわかる「就活 from A1」です。「筑波大の先輩の体験談を見る」ことができ、「一般的な就活情報サイト」と違って「自分の学類の先輩がどう就活したかがわかる機能」が備わっています。
